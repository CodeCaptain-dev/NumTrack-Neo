# NumTrack Neo — Version 1.0

**Next-gen number sorter with index tracking and high precision.**

---

## Overview

NumTrack Neo is a powerful number management and sorting tool designed for accuracy and traceability. Enter any amount of decimal numbers, and the program automatically sorts them in ascending order while keeping track of the original positions of each number. Its design makes it perfect for learning, analysis, and any scenario requiring precise number sorting.

This tool is built with double-precision floating-point support, giving you up to ~15–16 digits of accuracy. The program is optimized for performance, handling both small and large datasets efficiently, while maintaining a clean and user-friendly console interface.

---

## Features

- **Unlimited number input** (limited by system memory).  
- **Double-precision decimal support** (~15–16 digits).  
- **Automatic original index tracking**, so you always know the input position.  
- **Ascending sort output**, clear and easy to read.  
- **Console-based interface** with sleek startup header for a “device-like” feel.  
- Designed for **learning, experimentation, and analysis**.  
- Lightweight, fast, and minimal dependencies.  
- Fully traceable, ideal for debugging or educational use.

## Example Usage

Here’s a sample session:


Welcome to NumTrack Neo v1.0. \
Enter the number of numbers: 6 \
Number 1: 3.14 \
Number 2: 1.618 \
Number 3: 2.718 \
Number 4: 42.0 \
Number 5: 0.007 \
Number 6: 100.0 \

Sorted numbers: \
0.007, N5 \
1.618, N2 \
2.718, N3 \
3.14, N1 \
42.0, N4 \
100.0, N6    
Numbers loaded successfully...   \


**Notes on output:**

- `N0, N1…` indicate the **original input position**, starting at 0.  
- Works with decimals, integers, and mixed input.  
- Sorted output is in ascending order.  
- Original positions are preserved, giving full traceability.
